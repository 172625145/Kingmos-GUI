目录定义和组织：
kingmosroot-
           +-application(一般应用程序)
		   +-arch       (硬件相关层，hardware dependent layer）
		   +-build      (各种编译工程)
		   +-component  (为系统提供服务的组件，模块)
		   +-drivers    (设备驱动程序，硬件无关层)
		   +-filesys    (文件系统)
		   +-filerom    (文件系统ROM file)
		   +-include    (应用级头文件)
		   +-lib        (为各种平台或编译环境提供的二进制库文件)
		   +-library    (原代码级库文件)
		   +-private    (私有目录)
		   +-script     (脚本目录)
		   +-spec       (各种开发规格)
		   +-tools      (工具)
